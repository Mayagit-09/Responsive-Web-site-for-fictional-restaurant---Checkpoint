                                                                              O&M Restaurant

    - Présentation du projet
O&M Restaurant est un site web responsive développé avec **HTML5, CSS3 et Bootstrap 5**.  
Ce projet a été réalisé dans le cadre d’un checkpoint Bootstrap afin de pratiquer le **responsive design** et le système de grille.

---

    - Technologies utilisées
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons

---

    - Objectifs du projet
- Créer un site web responsive
- Utiliser le système de grille Bootstrap
- Concevoir une interface moderne et attractive
- Structurer une page web professionnelle

---

    - Fonctionnalités
- Navbar responsive avec menu mobile
- Section Hero avec image de fond
- Menu organisé par catégories (Entrées, Plats, Desserts)
- Cards Bootstrap avec images et descriptions
- Section À propos en deux colonnes
- Formulaire de contact
- Footer avec icônes sociales

---

    - Design
Le design est basé sur :
- un thème restaurant moderne
- des couleurs chaudes (jaune / noir)
- des animations simples au hover
- une mise en page propre et responsive

---

    - Aperçu du projet
Projet de restaurant fictif avec interface moderne et adaptée aux mobiles.
